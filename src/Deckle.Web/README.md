# Deckle.Web

Svelte 5 + SvelteKit web application for Deckle.

## Development

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
```

## Color Palette

- Dark Blue-Grey: `#35374B`
- Teal-Grey: `#344956`
- Muted Teal: `#50727B`
- Sage Green: `#78A083`

## API Configuration

The application automatically connects to the Deckle.API service when running through Aspire. The API endpoint is configured via environment variables set by the Aspire orchestrator.
